# 游戏日志分析知识库

本知识库用于记录和分析游戏道具日志相关的信息，包括完整的日志类型定义、字段含义、分析脚本和预设分析方案。

## 仓库内容

1. **完整日志类型分类** - 基于原始日志类型说明的完整分类
2. **日志类型更新记录** - 基于日志类型说明文档的补充更新记录
3. **分析脚本说明** - Python分析脚本使用说明
4. **分析方案** - 预设的分析方案和指标定义

## 快速开始

1. 克隆本仓库
2. 查看相关文档了解日志类型定义
3. 使用提供的分析脚本进行数据分析
