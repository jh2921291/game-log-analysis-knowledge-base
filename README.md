## 数据查询示例

### 查询特定渠道数据
```sql
-- 查询60001渠道的获得记录
SELECT * FROM item_table 
WHERE logtype = '60001' 
  AND item_num > 0
  AND substr(time, 1, 10) = '2026-03-25';

-- 查询70001渠道的消耗记录
SELECT * FROM item_table 
WHERE logtype = '70001' 
  AND item_num < 0
  AND substr(time, 1, 10) = '2026-03-25';
```

### 渠道对比分析
```sql
-- 对比核心产出渠道
SELECT logtype, COUNT(*) as count, SUM(item_num) as total
FROM item_table
WHERE logtype IN ('60001', '60008', '60012', '60015', '61108', '61160')
  AND item_num > 0
  AND substr(time, 1, 10) = '2026-03-25'
GROUP BY logtype
ORDER BY total DESC;
```
